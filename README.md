# Healthcare-Analytics-Case-Study-Excel-Power-BI
This project focuses on analyzing healthcare operational data to gain insights into patient flow, doctor performance, and overall hospital efficiency. The analysis combines Excel for data cleaning and preprocessing with Power BI for interactive dashboards and KPI visualization, enabling data-driven decision-making in a healthcare environment.

<br>
**Objectives**
<b>
Analyze patient demographics and visit patterns
<b>
Measure doctor efficiency and treatment time
<b>
Identify operational bottlenecks
</b>
Provide actionable business recommendations
<b></b>

**Tools & Technologies**
<b>
Microsoft Excel (Data Cleaning, Validation)
<b>
Power BI (Data Modeling, DAX, Visualization)
<b>
DAX Measures
<b>

**Key KPIs & Metrics**
<b>
Average Patient Age
<b>
Average Treatment Time
<b>
Doctor-wise Patient Count
<b>
Daily and Monthly Patient Trends
<b>
Department-wise Performance
<b>

**Business Recommendations**
<b>
Optimize doctor scheduling during peak hours
<b>
Improve resource allocation for high-demand departments
<b>
Monitor treatment time KPIs to enhance patient experience
<b>
